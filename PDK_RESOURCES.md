# PDK Resources — Curated Links

This file lists open source and public resources relevant to PDK development for 90–28nm nodes and adjacent processes.

## Key open PDKs and flow projects
- google/skywater-pdk — https://github.com/google/skywater-pdk (130nm) — widely used open PDK and reference for tooling.
- google/gf180mcu-pdk — https://github.com/google/gf180mcu-pdk (180nm) — useful for MCU-class flows.
- google/open-source-pdks — https://github.com/google/open-source-pdks — index of open PDKs.
- mflowgen/freepdk-45nm — https://github.com/mflowgen/freepdk-45nm (45nm) — compatible with mflowgen and OpenROAD flows.
- openecos-projects/icsprout55-pdk — https://github.com/openecos-projects/icsprout55-pdk (55nm)
- openecos-projects/icsprout55-pdk — https://github.com/openecos-projects/icsprout55-pdk
- The-OpenROAD-Project/asap7 — https://github.com/The-OpenROAD-Project/asap7 (academic PDK)
- ncsu-eda/FreePDK3 — https://github.com/ncsu-eda/FreePDK3

## EDA flows and toolchains
- The-OpenROAD-Project/OpenROAD — https://github.com/The-OpenROAD-Project/OpenROAD — RTL-to-GDS open toolchain.
- librelane — https://github.com/librelane/librelane — successor to OpenLane.
- SiEPIC (silicon photonics) — https://github.com/SiEPIC — SiEPIC tools and PDKs for photonics.

## Useful resources and references
- BlenderGDS — https://github.com/aesc-silicon/BlenderGDS — GDS importer with PDK support.
- Academic and vendor PDK repositories (search by node and vendor for more variants).
