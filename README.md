# ProcessDevelopmentKit
PDK resources
This repository collects resources, starter templates, and example flows to help with PDK development and integration in the 90–28nm node range.

Contents:

README.md (this file)
PDK_RESOURCES.md — curated list of open PDKs, flows, and vendor resources
docs/quickstart.md — quickstart to run a minimal validation (docs lint + link checks)
examples/simple-flow/Makefile — minimal example flow scaffold (non-functional placeholder)
.github/workflows/ci.yml — CI that checks docs and link health
CONTRIBUTING.md — contribution guidelines
LICENSE — MIT license
Target audience: EDA engineers, academics, and hobbyists who want a clean starting point for building or integrating PDKs for 90–28nm CMOS processes.
